## [1.0.0] — 2026-04-14

### Adicionado
- **SKILL.md**: Definição principal da skill com workflow de 4 fases (DISCOVER → MODEL → SPECIFY → VALIDATE).
- **README.md**: Documentação detalhada com casos de uso, princípios e tabela de recursos.
- **resources/implementation-playbook.md**: Playbook completo com padrões REST e GraphQL, exemplos de código, checklists e pitfalls comuns.
- **references/rest-best-practices.md**: Guia de referência de boas práticas REST (nomenclatura, status codes, idempotência, HATEOAS, segurança).
- **references/graphql-schema-design.md**: Guia de referência de design de schema GraphQL (tipagem, paginação Relay, mutations, N+1, deprecation).
- **references/api-versioning-strategies.md**: Guia de estratégias de versionamento de API e gerenciamento de breaking changes.

### Origem
- Baseado na skill community [api-design-principles](https://github.com/sickn33/antigravity-awesome-skills/tree/main/skills/api-design-principles) do repositório `antigravity-awesome-skills`.
- Adaptado e expandido para o padrão Skills Hub com workflow estruturado, recursos de referência em português e integração com o ecossistema local.

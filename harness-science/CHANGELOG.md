# Changelog: harness-science

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-27
### Added
- Initial release of the `harness-science` skill.
- **Ablation Methodology**: Standardized protocol for systematic capability testing.
- **Standard Flags**: Inclusion of `DISABLE_THINKING`, `DISABLE_BACKGROUND_TASKS`, and `SIMPLE_MODE` baselines.
- **Scientific Cycle**: Defined the Baseline -> Inject -> Ablate workflow.
- **Metrics**: Defined Success Rate (SR), Token Efficiency (TE), and Ablation Impact (AI).

# Token Distiller

Skill especializada em otimização de consumo de tokens através de compressão de contexto e modos operacionais de baixa/alta densidade.

## 🚀 Uso
- `/mode low` (ou `/caveman`): Ativa modo ultra-terse.
- `/mode high` (ou `/premium`): Ativa modo analítico completo.

Consulte [SKILL.md](SKILL.md) para detalhes técnicos.

# Changelog — Swarm Facilitator

All notable changes to this skill will be documented in this file.

## [1.2.0] - 2026-04-27
### Added
- **Technical Governance**: Integrated patterns for teammate synchronization and security.
- **Terminal Backends**: Added support/guidelines for iTerm2, Tmux, and In-Process runners.
- **Permission Bridge**: Formalized the propagation of guardrails from Leader to Teammates.
- **Resilience**: Added reconnection and heartbeat protocols.

## [1.1.0] - 2026-04-20
### Changed
- **Renaming**: Skill renamed from `multi-agent-orchestrator` to `swarm-facilitator` to avoid semantic collision with `sdd-orchestrator`.
- **Identity**: Updated to focus explicitly on choreography and persona handoff protocols.

## [1.0.0] - 2026-04-19
### Added
- Initial structure and definition of Swarm personas.
- Formal Handoff protocol.

# Feature Spec: Brainstorming Skill Enrichment (v1.2.0)

## Contexto
A skill de `brainstorming` atual é funcional, mas carece de técnicas estruturadas de engenharia de prompt modernas e padrões de design thinking consolidados. O objetivo é integrar o conhecimento da skill externa `brainstorming-skill` do GitHub, adaptando-a aos padrões do Hub.

## Requisitos
- [x] Analisar referências externas (Vanderbilt, Comprehensive Prompt Library).
- [x] Traduzir e adaptar técnicas para PT-BR.
- [x] Integrar o padrão "Flipped Interaction" na fase de Discover.
- [x] Criar catálogo de padrões (Perspective Multiplication, Inversion, Analogical Transfer).
- [x] Criar templates prontos para uso.
- [x] Atualizar workflow e versionamento da skill.

## Critérios de Aceitação (AC)
- **AC1**: A skill deve ter um `SKILL.md` atualizado com o novo workflow.
- **AC2**: Devem existir arquivos de referência em `references/` detalhando as técnicas.
- **AC3**: Deve haver um arquivo de `templates` em `resources/`.
- **AC4**: Toda a documentação deve estar em PT-BR.
- **AC5**: O CHANGELOG deve refletir a versão 1.2.0.

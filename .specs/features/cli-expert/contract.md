# Contract: New Skill - CLI Expert

## Agreement
The Implementer will create a brand new skill `cli-expert` that establishes the Hub's standard for designing high-maturity, AI-native CLI tools.

## Sensors (Validation)
1.  **Directory Presence**: Folder `/cli-expert` must exist.
2.  **Slash Command Pattern**: `SKILL.md` must mention the difference between `PromptCommand` and `LocalCommand`.
3.  **TUI Guidelines**: `SKILL.md` must include a section on using Ink/React for terminal interfaces.
4.  **Hub Audit**: `hb audit cli-expert` must pass.

# Skill Factory

The governance layer of the Agentic Hub. It ensures that every skill is built with high density, clarity, and structural integrity.

## How to use
When the user asks for a new skill:
1. Invoke `skill-factory`.
2. The factory will use `skill-factory-bootstrap` to create the structure.
3. Once the skill is developed, use `skill-factory-validator` to audit it.

## Standards
This factory enforces the **SDD v2.2.0** standard.

# Changelog: cli-expert

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-27
### Added
- Initial release of the `cli-expert` skill.
- **Slash Command Architecture**: Defined Prompt, Local, and Bundled command kinds.
- **TUI Standards**: Guidelines for building interactive terminal interfaces with React/Ink.
- **Metadata Registry**: Standardized command definition schema.
- **Context-Awareness**: Guidelines for command availability and LLM optimization.

# Onboarding Navigator Skill

Esta skill atua como um guia interativo e mentor cultural para o ecossistema do projeto. Ela transforma sua base de conhecimento estática (como o `GUIA_CENTRAL.md` e guias de decisão) em uma ferramenta ativa para orientar novos membros e a criação de novas funcionalidades, garantindo que os padrões Skynet sejam seguidos.

## Recursos

- **Navegação de Conhecimento**: Localiza e explica padrões do projeto rapidamente.
- **Suporte a Decisões Técnicas**: Guia o processo de escolha de tecnologias e criação de ADRs/RFCs.
- **Checklists de Onboarding**: Gera planos de ação para novos desenvolvedores ou módulos.
- **Mentoria Cultural**: Explica a lógica por trás das convenções do projeto.
- **Visualização**: Diagramas Mermaid para mapear processos e estruturas.

## Como Usar

Para ativar as diretrizes desta skill em sua sessão atual, utilize:

```bash
activate_skill onboarding-navigator
```

## Estrutura da Skill

- `SKILL.md`: Documento principal com o workflow de orientação.
- `references/`: Detalhes sobre tomada de decisão e estrutura de pastas.
- `examples/`: Guia de boas-vindas e fluxos de decisão visuais.

# Example: Interactive Welcome Message

Esta é uma amostra de como o `onboarding-navigator` deve se apresentar e orientar um novo usuário.

---

## 🤖 Boas-vindas ao Projeto Skynet!

Olá! Eu sou o **Onboarding Navigator**, seu guia interativo neste ecossistema. Minha missão é garantir que sua jornada por aqui seja produtiva, clara e alinhada com nossos padrões de excelência.

### 📍 Onde você está?
Você está no **AI Agent Skills Hub**, o repositório central de inteligência para nossos agentes. Aqui, cada diretório representa uma habilidade modular que capacita nossas IAs a resolver problemas complexos.

### 🚀 Por onde começar?
Para sua primeira sessão, recomendo seguir estes passos:

1.  **Reidratação de Contexto**: Peça-me para ler a pasta `.specs/project/` para entender nosso estado atual.
2.  **Entenda o Workflow**: Nossa lei é o **SDD** (`sdd/SKILL.md`). Nada entra no código sem ser especificado e planejado.
3.  **Setup de Ferramentas**:
    - Use `python-uv` para projetos Python.
    - Use `flutter-fvm` para projetos Flutter.

### 📋 Seu Checklist Inicial
- [ ] Ler o `GUIA_CENTRAL.md` (no repositório de docs).
- [ ] Configurar seu arquivo `.env` com as chaves necessárias.
- [ ] Executar `activate_skill sdd` para entender como criamos novas features.
